Recent Projects

UAV Fire Detection

In the Fall 2023 Semester, I joined the UAV Fire Detection Project sponsored by Lockheed Martin at Cal Poly Pomona. As the Machine Learning Team Lead, I was tasked with coordinating the ML team members and finding ways for us to collaborate and achieve our goal: our autonomous UAV being able to identify wildfires using machine learning. Using the YOLOv8 software, we were able to train a model to recognize fires with an accuracy rate of about 80%. Through this process, I practiced and built upon my teamwork, leadership, and communication skills, providing updates to the Project Lead in a timely manner. Additionally, I helped present our team's progress and implementation plans to Lockheed Martin represenatives.

Boston Dynamics SPOT Project

In Spring 2023, I joined the CPP Boston Dynamics SPOT Project, where we programmed SPOT to autonomously deliver packages. Using fiducial markers (sort of like QR codes), SPOT could locate and move towards a package, pick it up using its arm, and deliver it to another marked location. We coded in Python and used the Boston Dynamics SDK to help bridge the gap between the software and hardware components of this robotics project. We presented our project at the College of Science Research Symposium as well as at an Open House for new students.

Link to GitHub repository: https://github.com/Samanyu24X/SPOT-PROJECT

Flashcard Mobile App

In this project, I designed and created an Android app for Flashcard decks using Java and Android Studio. With my app, users can create, edit, and save their own flashcard sets. When quizzing themselves with the flashcards, they can either click on the card to flip it and reveal the result, or pick one of the three possible answers below. I imported libraries that allowed me to implement special effects, and added an icon for the app itself. I updated and added features to the app over the course of four weeks, and used an Android emulator on my machine to run testcases and debug.

Link to GitHub repository: https://github.com/Samanyu24X/SamanyuSatheeshFlashcardApp
